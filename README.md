## Overview

This repository contains the source code for the professional web platform of **Javier Clot**. The application is engineered to serve as a high-prestige digital headquarters, providing prospective clients and colleagues with a comprehensive overview of the attorney's legal trajectory, specialized expertise, and professional philosophy.

The platform is designed to establish institutional credibility, offering a streamlined interface to navigate through various legal disciplines and consultation services provided by the practitioner.

---

## Key Features

* **Professional Biography:** A dedicated section detailing the academic background, career milestones, and legal certifications of Javier Clot.
* **Specialized Practice Areas:** Organized modules outlining expertise in various legal fields, providing clarity on the scope of representation offered.
* **Foundational UX/UI:** Utilizing a minimalist and authoritative design language to reflect professional integrity.
* **Optimized Performance:** Integration of `next/font` for superior loading of the Inter typeface and refined system performance.
* **Mobile-First Architecture:** Full responsiveness across all modern device categories to ensure accessibility for every client.

## Getting Started

To execute the project in a local development environment, ensure you have **Node.js** installed.

### 1. Dependency Installation

Install the required packages using your preferred manager:

```bash
npm install
# or
yarn install
```

### 2. Local Execution

Launch the development server:

```bash
npm run dev
```

The application will be accessible at `http://localhost:3000`. You may modify the core application logic by editing the `app/page.tsx` file.
